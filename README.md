# GA Workflow Center

`GA_업무자동화툴_기획서.md`의 1차 범위를 구현한 **단일 파일 웹앱**이다. 서버·DB·외부 라이브러리 없이 `GA-Workflow-Center.html` 하나로 동작하며, 모든 기록은 사용자 브라우저의 localStorage에 저장된다.

## 배포 방법 (팀 허브 포털)

사내 팀 허브 포털에 올릴 때는 다음 중 하나를 사용한다.

1. **파일 업로드/붙여넣기**: `GA-Workflow-Center.html` 내용 전체를 복사해 포털의 도구 등록(HTML) 기능에 붙여넣는다. 외부 의존성이 전혀 없어 그대로 동작한다.
2. **정적 파일 배치**: 포털 서버의 정적 경로에 `GA-Workflow-Center.html`을 올리고 링크로 등록한다. (예: `/tools/ga-center/GA-Workflow-Center.html`)
3. **로컬 실행**: 파일을 더블클릭해 브라우저로 바로 열어도 동일하게 동작한다.

주의. 데이터는 **접속한 브라우저별로 따로 저장**된다(개인 업무 보조 도구 — 기획서 12-6 결정과 일치). PC를 옮길 때는 우상단 [데이터 내보내기]로 JSON 백업 후 [가져오기]를 사용한다.

## 금액 기준 참고

품의 기준 금액은 **30만원**이다(사용자 확정, 2026-07-09). 원본 마스터리스트·지출결의 공통노하우에는 20만원으로 적힌 곳이 있으나 30만원이 최신 기준이다. 변경 시 `GA-Workflow-Center.html`의 `PO_THRESHOLD` 상수 하나만 수정하면 된다.
